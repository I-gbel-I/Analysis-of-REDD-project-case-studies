Analysing forest loss for two REDD projects in Brazil and Cambodia: the Agrocortex REDD poject, and the Tumring REDD+ project;
using the Hansen global forest change dataset, and Sentinel 2A imagery.


Data sources:

Agrocortex REDD project : https://registry.verra.org/app/projectDetail/VCS/1686
Tumring REDD+ project: https://registry.verra.org/app/projectDetail/VCS/1689

Hansen Global Forest Change 2000-2022: https://storage.googleapis.com/earthenginepartners-hansen/GFC-2022-v1.10/download.html
Sentinel 2A imagery: https://scihub.copernicus.eu/
